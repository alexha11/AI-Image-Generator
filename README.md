# AI Image Generator

## Description

This is a creating and sharing images platform built using MERN stack (MongoDB, Express, React, Node.js). It allows users to create image based on their props and share them to the community. (To create image, users have to login in their account) 

## Live Deployment

Live demo:  [AI Image Generator website]() 

The server is deployed [AI Image Generator Server](https://ai-image-generator-f5m8.onrender.com).

## Warning

- The live demo may get a bit slow since it is a free plan
- The OpenAI api key is a paid plan so you have to login to create an image and each account can create up to two images

## Run Locally

- Clone the repository
- Set up `.env` file in `server` directory with the following content:

```bash
MONGODB_URL=<your_mongodb_url>
OPENAI_API_KEY=<your_api_key>
PORT=8080

CLOUDINARY_CLOUD_NAME=<user_name_from_cloudinary>
CLOUDINARY_API_KEY=<api_key_from_cloudinary>
CLOUDINARY_API_SECRET=<api_key_secret_from_cloudinary>

```

- Run `npm install` in both `client` and `server` directories
- Run `npm dev` in both `client` and `server` directories
